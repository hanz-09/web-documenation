Get Started

Instal Git: Make sure you have Git installed on your system. If you don't, you can download it from git-scm.com and follow the installation instructions.

Install Flutter: Ensure that Flutter is installed on your machine. You can download it from flutter.dev and follow the setup instructions for your operating system.

Installation

1. Open Terminal: Open your terminal (Linux/macOS) or command prompt (Windows).

2. Clone the Repository: Suppose the GitHub repository URL is github.com. Then, run the following command: 

              
                git clone https://github.com/hanz-09/pbl.git

3. Navigate to the Project Directory: 

              
                cd flutter

4. Set Up Firebase: Follow the steps above to create a Firebase project and download the necessary configuration files (google-services.json for Android and GoogleService-Info.plist for iOS).

5. Add Configuration Files: Place the configuration files in the appropriate directories (android/app and ios/Runner).

6. Update Dependencies and Initialize Firebase:Ensure your pubspec.yaml includes Firebase dependencies.Initialize Firebase in your main.dart.

7. Get Flutter Packages: 

              
                flutter pub get

8. Run Flutter App: 

              
                flutter run

Configuration

Configuring Firebase in a Flutter project involves several steps. Below is a detailed guide to integrate Firebase into your Flutter app for Android platforms:

1. Create a Firebase Project: Go to the Firebase Console. Click on "Add project" and follow the on-screen instructions to create a new project.

2. Register Your App with Firebase:

  • In the Firebase Console, go to Project Settings.
  • Click on the Android icon to add an Android app
  • Register your app with the package name (e.g., com.example.yourapp).
  • Download the google-services.json file.
  • Place the google-services.json file in the android/app directory of your Flutter project.

3. Add Firebase SDK Dependencies

  • Open your Flutter project's pubspec.yaml file and add the necessary Firebase dependencies:
    •               
                dependencies:
                  flutter:
                    sdk: flutter
                  firebase_core: latest_version
                  firebase_auth: latest_version
                  cloud_firestore: latest_version
                  # Add other Firebase dependencies as needed
  • Run flutter pub get to install these dependencies.

4. Configure Your Android Project

  • Open android/build.gradle and add the Google services classpath:
    •               
                buildscript {
                  dependencies {
                    classpath 'com.google.gms:google-services:4.3.10' // Check for the latest version
                  }
                }
  • Open android/app/build.gradle and apply the Google services plugin:
    •               
                apply plugin: 'com.google.gms.google-services'

5. Initialize Firebase in Your Flutter App

  • Open your main.dart file and initialize Firebase:
    •               
                import 'package:firebase_core/firebase_core.dart';
                import 'package:flutter/material.dart';

                void main() async {
                  WidgetsFlutterBinding.ensureInitialized();
                  await Firebase.initializeApp();
                  runApp(MyApp());
                }

                class MyApp extends StatelessWidget {
                  @override
                  Widget build(BuildContext context) {
                    return MaterialApp(
                      home: Scaffold(
                        appBar: AppBar(title: Text('Ceker')),
                      ),
                    );
                  }
                }
